# Calculadora Console

Esta é uma simples calculadora de linha de comando desenvolvida em C# que permite realizar operações matemáticas básicas, como adição, subtração, multiplicação e divisão.

## Funcionalidades

- Adição: realiza a soma de dois números.
- Subtração: realiza a subtração de dois números.
- Multiplicação: realiza a multiplicação de dois números.
- Divisão: realiza a divisão de dois números. (Observação: o divisor não pode ser zero).

## Como Usar

1. Baixe ou clone este repositório em seu computador.
2. Abra o projeto em seu ambiente de desenvolvimento preferido.
3. Compile e execute o projeto.
4. Selecione uma opção de operação desejada digitando o número correspondente:
   - 1 para adição.
   - 2 para subtração.
   - 3 para multiplicação.
   - 4 para divisão.
   - 5 para fechar o programa.
5. Siga as instruções na tela para inserir os números necessários para a operação selecionada.
6. O resultado será exibido na tela.

## Requisitos

- .NET Core SDK 3.1 ou superior.
- Um ambiente de desenvolvimento compatível com C# (como Visual Studio, Visual Studio Code, etc.).
